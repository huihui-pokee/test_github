About the 24 Game

The 24 Game is a mathematical card game where the objective is to find a way to manipulate four numbers to get the result of 24. You can add, subtract, multiply and divide, using each number exactly once.

This classic game was invented in the 1960s and has been used in schools worldwide to help students improve their mental math skills and arithmetic fluency.

Playing the 24 Game regularly can help improve:

  • Mental calculation speed
  • Number sense and fluency
  • Problem-solving skills
  • Mathematical creativity
